Caja

Es la forma en que CSS ve a los elementos HTML, los ve como si fueran cajas: 

        
            h1{
                border: 1px solid red;
                padding: 5px;
                margin: 25px;
            }

Y tienen las siguientes propiedades

  • Contenido: Tamaño del elemento ancho y alto (442x37) todo lo que ocupa el elemento en el viewport. Cada que se redimenciona la ventana cambia este tamaño
  • padding: relleno entre el borde y el elemento
  • Border: borde que encierra el elemento.
  • Margin: distancia entre el borde y los demas elementos

Dentro del modelo de caja se consideran 4 lados:

  • Top
  • Right
  • Button
  • Left
        
            p{
                border: 3px solid blue;
                padding: 8px 10px 7px 15px;
                margin: 14px 20px;
            }

Propiedad Display

La propiedad display es el comportamiento de visualización de los elementos. Puede tomar los siguientes valores

  • none: quita el elemento
  • inline: elemento de linea, uno junto a otro
  • block: elemento de bloque, uno debajo del otro
  • inline-block: comportamiento de linea pero deja modificar el ancho y el alto

Tamaño de caja

Dependiendo del tipo de caja, si es de content o border va a tomar el tamaño. Cuando es de tipo content, al ancho y alto le suma el padding y el border. Cuando es border el ancho y el alto siempre va a ser el mismo sin importar el padding y el border
